### Building/Testing
This project was developed originally using MSVC and CMake. However, this should also work in GCC, Clang, etc.  
Please mainly use CMake for building and testing as it is the way we intended.

### Code conventions
- 1 tab indentation is equal to 4 spacebar presses
- File including/importing will be ordered as follow:
	1. Standard library (STL)
	2. External libraries/dependencies
	3. User-defined libraries
	4. Project's files
- Function names should start with capital letter and are easy to understand
